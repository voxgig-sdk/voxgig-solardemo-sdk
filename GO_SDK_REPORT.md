## Issues Found

### 1. BLOCKING: Duplicate `TestExists` function — compilation failure

**File:** `go/test/projectname_sdk_test.go`

**Error:**
```
test/projectname_sdk_test.go:9:6: TestExists redeclared in this block
    test/exists_test.go:9:6: other declaration of TestExists
FAIL    voxgigsolardemosdk/test [build failed]
```

**Cause:** The SDK generator produces two files that both declare `func TestExists`:
- `go/test/exists_test.go` — generated by the `cmp/go/TestEntity_go.ts` component
- `go/test/projectname_sdk_test.go` — copied from the `.sdk/tm/go/test/projectname_sdk_test.go` template

Both files are identical in content. The template file `projectname_sdk_test.go` in `.sdk/tm/go/test/` contains a `GOMODULE` placeholder and is designed to be merged/renamed during generation, but the merge logic produces both output files instead of one.

**Impact:** `go test ./...` fails to compile entirely. No Go tests can run.

**Workaround:** Remove or rename `go/test/projectname_sdk_test.go`. With that file excluded, all tests pass.
